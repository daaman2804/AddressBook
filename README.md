# Address Book

A command-line Address Book application developed in C that allows users to create, search, edit, delete, and manage contacts efficiently. The project demonstrates practical usage of data structures, file handling, input validation, and modular programming concepts.

---

## Features

- Create new contacts
- Search contacts by name
- Edit existing contact details
- Delete contacts
- Display all saved contacts in sorted manner (name/phone/email)
- Prevent duplicate phone numbers and email addresses
- Validate phone numbers and email formats
- Persistent storage using CSV files
- Menu-driven user interface

---

## Technologies Used

- Language: C
- Compiler: GCC
- File Storage: CSV
- Libraries:
  - stdio.h
  - string.h
  - ctype.h
